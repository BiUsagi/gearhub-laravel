## 🎯 Chức năng chính

### 👤 Người dùng (Customer)

- [ ] Đăng ký / Đăng nhập / Quên mật khẩu
- [ ] Đăng nhập bằng mạng xã hội (Google, Facebook)
- [ ] Quản lý hồ sơ cá nhân
- [ ] Thêm nhiều địa chỉ nhận hàng
- [ ] Thêm sản phẩm vào giỏ hàng
- [ ] Tạo đơn hàng, theo dõi trạng thái
- [ ] Thêm sản phẩm vào mục yêu thích (wishlist)
- [ ] Xem danh mục, lọc theo thương hiệu, tag, tìm kiếm
- [ ] Xem chi tiết sản phẩm, hình ảnh, mô tả, đánh giá
- [ ] Viết đánh giá sản phẩm
- [ ] Áp dụng mã giảm giá (coupon) khi thanh toán
- [ ] Liên hệ / gửi yêu cầu hỗ trợ

---

### 🛒 Đơn hàng

- [ ] Tạo đơn hàng từ giỏ
- [ ] Lựa chọn phương thức giao hàng
- [ ] Thanh toán COD, chuyển khoản, ví điện tử (Momo, VNPAY - sau)
- [ ] Theo dõi trạng thái đơn hàng
- [ ] Lưu lịch sử đơn hàng

---

### 🛠️ Quản trị viên (Admin)

- [ ] Dashboard tổng quan
- [ ] Quản lý người dùng, phân quyền (Spatie)
- [ ] Quản lý danh mục, thương hiệu
- [ ] Quản lý sản phẩm & ảnh sản phẩm
- [ ] Gắn tag, gán giảm giá
- [ ] Quản lý đơn hàng (xác nhận, giao hàng, huỷ)
- [ ] Quản lý đánh giá, phê duyệt trước khi hiển thị
- [ ] Quản lý banner quảng cáo
- [ ] Quản lý trang nội dung (About, Chính sách,...)
- [ ] Quản lý mã giảm giá
- [ ] Quản lý ticket hỗ trợ
- [ ] Cài đặt website (logo, tên, hotline,...)
- [ ] Nhật ký hoạt động (logs)
